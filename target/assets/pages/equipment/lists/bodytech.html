Each bodytech entry has seven listed characteristics:

Name: Each piece of bodytech has a unique name, making it easier for you to find the augmentation.

Description: Each piece of bodytech provides a short description, and a detailed description; the short description in the tables below provide you with a short overview of the pieces general functions, while the detailed description provides you with a more in-depth description of it's functions, a list of available modifications, and general fluff.

Category: The category of an augmentation is a vague description of the overall type and location of the cybernetic. Usually, only one type of cybernetic can be installed per category, but this can be overruled (with the Gamemasters' permission). As a rule of thumb, bodytech generally replaces existing tissue, bones and muscle, so it doesn't make that much sense for a character to have two cybernetic spines, for example. Consider the description of the piece of 'ware when determining if and where a character can have it installed.

Essence: Each piece of 'ware has an Essence cost; when installing a piece of bodytech, this is the amount of Essence Loss a character suffers. Essence Loss is generally permanent, but uninstalling cybertech leaves an Essence Hole in the character, that can be filled with new cybertech. Most character have an Essence Loss Threshold of 7, and can only receive 6 Essence Loss without dying. Some bodytech causes more Essence Loss than others, and some list available modifications that cause additional Essence Loss, so keep that in mind when attending the next chop shop.
